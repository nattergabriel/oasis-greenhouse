## Summary of decisions made

| Decision | Before audit | After audit | Rationale |
|---|---|---|---|
| **Primary brand color** | Terracotta (#E2725B) | Greenhouse Green (#4CAF50) | We're an agriculture project. Green = growth. Syngenta alignment. Research consensus for ag-tech. |
| **Background** | #070B34 (purple-tinted dark) | #0A1628 (true dark navy) | More premium, more cinematic. Better card contrast. Michael's preference. |
| **Slide approach** | All slides identical dark navy | Cinematic hero (1,5,7) + clean dark content (2,3,6) | Syngenta enablement deck model. Creates visual rhythm. Fixes "monotone = standard" problem. |
| **Imagery** | None (flat colored rectangles) | Photorealistic Mars greenhouse (inspired by Syngenta enablement deck) | Research: "imagery does the heavy lifting." The enablement image bridges Mars + agriculture naturally. |
| **Visual motif** | None | Terrain contour lines + sweeping curve element | Inspired by Syngenta's curved green band. Contour lines work for both Mars terrain and agricultural land. |
| **Logo** | None | TERRA NOVA wordmark + leaf icon in green | Research: "essential for hackathon branding." Simple, memorable, 3 or fewer colors. |
| **Color count** | 13 | 6 core + 5 extended (dashboard only) | Research: "2-4 main colors." Simplified palette. |
| **Headlines** | 24pt | 36-72pt | Research: "bold typography as hero, 60-72pt+" for impact. |
| **Demo** | Static dashboard mockup | 60-90 sec cinematic video with music | Michael's vision: epic music, astronaut briefing, mission timeline. |
| **Total slides** | 7 all same style | 3 cinematic + 3 content + 1 demo transition | Creates rhythm: emotional → informational → emotional. |

## Design direction: cinematic realism

Inspired by the Syngenta/AWS enablement session deck. Their hero slide features a photorealistic 3D-rendered Mars greenhouse (glass dome, green plants, astronauts, red terrain, starry sky) with a sweeping white-to-green curved brand element. Content slides switch to clean light backgrounds.

Our adaptation: **cinematic Mars imagery for hero slides (1, 5, 7), dark navy for content slides (2, 3, 4, 6).** We stay dark (not light backgrounds) because our project is tech/space-forward and the dashboard is dark-themed. But the hero slides bring the visual drama and emotional impact.
