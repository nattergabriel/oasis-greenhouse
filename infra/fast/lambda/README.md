# Lambda Authorizer for API Gateway

This Lambda function validates the `X-API-Key` header for all API Gateway requests.

## Files

- **index.js** — Lambda handler function
- **package.json** — Dependencies
- **build.sh** — Build script to create deployment package

## Build

The Lambda deployment package must be built before running Terraform:

```bash
cd infra/fast/lambda
./build.sh
```

This creates `authorizer.zip` which Terraform deploys.

## How It Works

1. Extracts `X-API-Key` from request headers
2. Fetches expected API key from AWS Secrets Manager (cached)
3. Returns `{ isAuthorized: true/false }`

## Files NOT in Git

- `node_modules/` — installed by `npm install`
- `authorizer.zip` — created by `build.sh`
- `package-lock.json` — auto-generated by npm

These are excluded via `.gitignore` and rebuilt during deployment.
