# Frontend

This directory is the frontend project for the hackathon. No framework has been chosen yet - pick one when the hackathon starts (React, Next.js, Vite + React, etc.).

## Available Skills

The `.claude/skills/` directory contains 9 reference skills for rapid frontend development:

- **react-patterns** - Hooks, custom hooks, component patterns, context, state management
- **tailwind-css** - Utility class reference for layout, spacing, typography, responsive design
- **ui-components** - Copy-paste-ready components: modals, dropdowns, tabs, toasts, tooltips, etc.
- **forms** - Form handling, validation, file uploads, multi-step forms
- **api-patterns** - Fetch API, data fetching hooks, error handling, caching, WebSocket/SSE
- **design-system** - Color palettes, typography, spacing scale, shadows, CSS variables
- **animations** - CSS keyframes, Framer Motion, loading spinners, page transitions
- **data-viz** - Recharts, Chart.js, D3.js, dashboard stat cards, data formatting
- **accessibility** - ARIA attributes, keyboard navigation, focus management, screen readers

These skills are automatically loaded by Claude when relevant to your work. You can also invoke them directly with `/skill-name`.

## Conventions

- Use Tailwind CSS for styling unless the team decides otherwise
- Follow the patterns from the skills for consistency
- Keep components small and focused
- Use TypeScript if the team is comfortable with it
